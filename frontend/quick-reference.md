# Quick Reference

- Types:
  - `Int`
  - `Bool`
  - `Top`
  - `Bot`
  - `forall a. Type`
  - `forall (a <: Type). Type`
  - `Type -> Type`
  - `Type & Type`
  - `Type | Type`
  - `(Type, Type)`
- Terms:
  - Int literals: `0`, `1`, `2` ...
  - Bool literals: `True` / `False`
  - Lambda: `\x. x`
  - Application: `(\x -> x) 1`
  - Type annotation: `1 : Int`
  - Type abstraction: `/\a. (\x -> x) : a -> a`
  - Type application: `(\a. (\x -> x) : a -> a) @Int 3`
  - Let: `let id = \x. x in id 1`
